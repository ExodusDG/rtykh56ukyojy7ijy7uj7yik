Метод createTask - Создание нового задания

Параметры запроса:

Параметры

Описание

api_token string , обязательно

Токен авторизации

method string , обязательно

Метод запроса

version float , обязательно

Номер версии api. Текущая версия 1.0

website string , обязательно

Соц. сеть, актуальный список можно получить вызвав метод getServices

type string , обязательно

Тип накрута, актуальный список можно получить вызвав метод getServices

quality string , обязательно

Качество, актуальный список можно получить вызвав метод getServices

link float , обязательно

Ссылка на объект задания

count integer , обязательно

Количество

comments array of objects

Массив вариантов комментариев, если не передан, то используются стандартные комментарии в зависимости от типа и качества услуги

Успешный ответ

{

response:

{

"id": 100,

"link": "https://vk.com/id1",

"price": 347.52,

"balance": 10534.0,

"status": "Выполнено"

}

}

Ошибки

Тип

Описание

MISSING_PARAMETER

Переданы не все параметры

INVALID_API_TOKEN

Нерабочий API ключ

INVALID_VERSION

Неверная версии api

INVALID_METHOD

Неверный метод

COUNT_FOR_THIS_ORDER_TOO_LOW

Количество меньше минимального

INVALID_AMOUNT_OF_ORDER

Минимальная цена заказа 1 руб

Недостаточно средств на балансе

URL_INVALID

Неверная ссылка

RULES_ERROR

Контент задания нарушает правила

API_TOKEN_DISABLED

Взаимодействие с API ключом выключено

ACCOUNT_BLOCKED

Аккаунт заблокирован

INVALID_COUNT

Количество должно быть целым числом

CREATE_ORDER_ERROR

Ошибка создания заказа

Цены и ограничения по количествам на каждую услугу изменяются не зависимо от версии api, если вы хотите быть заранее проинформированы о важных изменениях следите за нашим телеграмм каналом @EasyLiker

Метод getTasks - Получение списка заданий

Параметры запроса:

Параметры

Описание

api_token string , обязательно

Токен авторизации

method string , обязательно

Метод запроса

version float , обязательно

Номер версии api. Текущая версия 1.0

id integer

Номер заказа. В ответе будет один заказ в массиве. Если не передан, то вернется count последних заказов

count integer

Количество заказов, которые вернет сервер сортируя от новых к старым. По умолчанию 20

offset integer

Смещение по вашим заказам. По умолчанию 0

Успешный ответ

{

response:

[{

"id": 100,

"date": "02.01.2021 | 04:20:34",

"link": "https://vk.com/id1",

"count": 1450,

"sum": 217.5,

"name": "VK друзья | офферы, низкое качество"

"status": "Выполнено"

}]

}
